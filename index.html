画像の読み込み、image()関数

サンプルコードの結果

サンプルコード

　

　　 PImage img1;

　　 PImage img2;

　　 void setup(){

　　 　size(480, 120);

　　 　img1=loadImage("lunar.jpg");

　　 　img2=loadImage("capsule.jpg");

　　 ｝

　　

　　 void draw(){

　　 　image(img1, -120, 0);

　　 　image(img1, 130, 0, 240, 120);

　　 　image(img2, 300, 0, 240, 120);

　　 }　　

説明文

　画像を表示させるためには、次の3ステップをまず行う。

　①detaフォルダに画像を追加する。（画像データをコード画面にドロップ）

　②画像を格納するためにPImage変数を作る。

　③loadImage()を使ってその変数に画像を読み込む。

　　

　前述で画像データを読み込んだらimage()関数を用いて表示させる。

　image(a, b, c, d, e);

　a:読み込んだ画像の変数を指定する。

　b:x座標

　c:y座標

　d:表示時のサイズの幅

　e:表示時のサイズの高さ

　dとeは必須項目ではなく、オプションである。指定しない場合は画像本来のサイズで表示される。

　例では、複数の画像を読み込みサイズを変更することですべて表示させている。

　※画像は例として授業時に配られたものを使用する。
